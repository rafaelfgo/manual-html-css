[id] - [class]

Essas tags tem função de indentificar outras tags e poder aplicar css especifico para uma tag genérica.
[id] - apenas um único elemento deve receber o nome da id.
[class] - um conjunto de elemtos podem receber a mesma classe.

<p id="principal">
    Texto aqui.
<p/>

<p class="secundario">
    Texto aqui.
<p/>

<p class="secundario">
    Texto aqui.
<p/>


A aplicação no CSS

[id] - ao se referir ao id substitua por: #
[class] - ao se referir a class substitua por: . 

<style>
    #principal{
        background-color: white;
    }
    .secundário{
        background-color: blue;
    }
<style/>
