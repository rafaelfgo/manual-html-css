Audio e Source


Posso colocar vários tippos de arquivos de áudio para o tipo que melhor abrir no navegador.
src - a localização do arquivo.
type - o formato do arquivo.    
    audio
        source src="____" type="_____"
        source src="____" type="_____"
        source src="____" type="_____"
    audio

Alguns parâmetros importantes para colocar dentro da tag aúdio:
- control - aparecer os controles

- preload - funções associadas ao carregamento do arquivo
  - auto - ele só vai considerar que o site terminou de carregar se o audio estiver carregado todo também. [vem como padrão]
  - metadata - ele carrega informações básicas do aúdio, antes do usuário apertar o play [recomendado]
  - none - ele não carrega nada do arquivo de aúdio até o usuário apertar o play.

- loop - o áudio ao chegar no final volta para o início.

auido - Type

O type é para informar o tipo de arquivo e cada arquivo possui uma forma especifica. Pode ser conferido em alguns site.

audio controls preload="metadata"
    source src="pasta/well.mp3" type="audio/mpeg"
    source src="pasta/well.ogg" type="audio/ogg"
    source src="pasta/well.wav" type="audio/wav"
/audio
