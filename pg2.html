As formas de representar as cores

Existem 4 formatos para usar as cores no CSS3, são eles:
hexadecinal #0000ff
hsl - (cor, saturação, luminosidade)
rgb - (red, green, blue)
rgba - tem a opção de deixar transparente

Como descobrir a numeração de uma cor

salve a imagem e abra no photoshop depois use a ferramenta que coleta a cor para ver a numeração

Como fazer degradê no CSS

Para fazer o degradê use a tag:

background-image: linear-gradient(_______);

Dentro do gradient a direção pode variar os lados. Pode ser escrito como "right,left" e como "ângulos". Exemplo:
background-image: linear-gradient(to right, white, blue);
background-image: linear-gradient(to left, white, blue);
background-image: linear-gradient(to top, white, blue);
background-image: linear-gradient(to bottom, white, blue);

background-image: linear-gradient(to right, white, blue, purple)
background-image: linear-gradient(180deg, rgb(207, 26, 26), blue, purple);
