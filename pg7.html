[tag filho]

É usado o sinal de "menor que" para indicar que todas tags X dentro daquela tag Y devem receber aquela configuraçãode CSS.

<style>
  main < p{
        font-size: 16px;
        color: black;
        font-family: Arial;
    }    
</style>
    
[tag grupo]

É usado para indicar que todas as tags dentro da primeira tag, independente de estarem mais internas contuam recebendo aquela configuração. Não se usa sinal apenas espaçamento.

<style>
  main  p{
        font-size: 16px;
        color: black;
        font-family: Arial;
    }    
</style>
