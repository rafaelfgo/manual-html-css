A anatômia das letras

serifa - são as pontas que aparecem nas letras. Serifa serve para criar linhas imaginárias, como ligasse com palavras. Impressões em papel é recomendado o uso de serifa.
san-serif - fonte sem serifa. Especialistas falam que em textos muitos longos em impressões virtuais dÊ preferência a textos não serifados.
monoespaçada - essas fontes deixam o mesmo espaçamento entre as letras.

para o tamanho da letra o mais recomendado é usar:
medida absoluta: pixel (px)
medida relativa: (em)

Normalmente 16 px é o tamanho normal da fonte.
font-size: 16px;

Como trabalhar com medida relativa?
A fonte de tamanho relativa é bom pois vai depender do tamanho da tela. Por esse motivo é melhor usar em como tamanho. 1em = 16px.
font-size: 1em;

Peso da letra

existem letras específicas que possuem variações de peso. Após ter escolhido a letra que possue peso posso escolher a intensidade desse peso. Com: [lighter] [normal] [bold] [bolder]
font-family: 'work Sans';
font-weight: bolder;

Obs.: algumas fontes tem variações para weight, com variações numericas de 100 a 900.

Fonte style

font-style: italic;

Fonte sublinhado

text-decoration: underline;

Encurtando o código

font-style: italic;
font-weight: bold;
font-size: 1em;
font-family: 'work Sans';

Para simplicar o código posso escrever o seguinte obedecendo a ordem de cima:
font: italic bold 1em 'work Sans', sans-serif;
