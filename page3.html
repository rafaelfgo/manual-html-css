Introdução

É importante saber que vídeos pesam e por isso devemos ter cuidado com hospedar vídeos no próprio site.
Algumas possibilidades para colocar vídeos no site é ter um outro local para hospedar os sites. 

    
Converter vídeos

É importante ter algum programa para converter vídeos em formatos diferentes, caso eu hospede no meu próprio site, pois pode ser que o servidor do usuário 
não consiga abrir em certo formato, além do peso do arquivo. Um programa recomendado é o HANDBREAK.
    
Vídeos Internos

1. Forma - A mais simples com a tag [video] apenas.
<video src="________"></video>

1.1 Lembre-se do parâmetro [controls] para aparecer os controles.
<video src="________" controls></video>

1.2 Posso modificar o tamanho com o parâmetro [width]
<video src="________" controls width="______"></video>

1.3 Posso adicionar outro parâmetro chamado [loop].
<video src="________" controls width="______" loop></video>

2. Forma - com vários formatos para abrir em todos navegadores.
<video controls width="300px">
    <source src="midia/meu-video.mp4" type="video/mp4">
    <source src="midia/meu-video.m4v" type="video/mp4">
    <source src="midia/meu-video.webm" type="video/webm">
    <source src="midia/meu-video.ogv" type="video/ogg">
</video>
Para saber o type de cada vídeo procure na internet.
    
Videos Externos

1. Através da tag [iframe]. Só copiar e colar o iframe de onde está o vídeo.
<iframe> width="560" height="315" src="https://www.youtube.com/embed/3hng-hmSv2Y" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; 
clipboard-write; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>
