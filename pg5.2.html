[background-image] [Border] [display] shorthand

Página inicial 

Comando para criar as boxes e colocar uma do lado da outra
<style>
div.quadrado{
    background-color: gray;
    border: 2px solid black;
    display: inline-block;
    width: 300px;
    height: 300px;
}
#q1{
    background-color: rgb(255, 230, 200);
}
#q2{
    background-image: linear-gradient(to right, yellow, blue);
}
#q3{
    background-image: url(imagens/fotodefundo.png);
}
</style>
--------------
<body>
    <div class="quadrado" id="q1"></div>
    <div class="quadrado" id="q2"></div>
    <div class="quadrado" id="q3"></div>
