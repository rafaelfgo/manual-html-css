Tabelas em HTML5

As tabelas em HTML5 são úteis para formar tabelas que possuem uma importância semântica e, por isso, não devem ser usadas
para fazer a estrutura do site em si.

Estrutura básica das tabelas



    

    Table - é a tabela
    tr - a linha da tabela
    th - elementos dentro da tabela (colunas)
    
    <table>
        <tr>
            <td>A1</td>
            <td>B1</td>
            <td>C1</td>
        </tr>
    </table>
    

    

        

            

                A1
                B1
                C1
            

            

                A2
                B2
                C2
            

        

    




Alinhamento dentro da tabela - CSS3

td{
    text-align: center; (alinhamento horizontal)
    vertical-align: top; (alinhamento vertical)
    vertical-align: middle;
    vertical-align: bottom;
}

Linhas na tabela - CSS3

Para colocar as linhas é só colocar as bordas, porém o comando por padrão tem espaços, nesse caso devemos usar um comando específico.

table{
    border-collapse: collapse;
}
td, th{
    border: 1px solid black;
}

Cabeça, Corpo, Pé da tabela

Posso adicionar elementos na tabela que não são dados mas áreas que dizem o que são os dados abaixo.


    

th - informação em relação aos dados
td - dado da tabela
thead - cabeçalho da tabela
tbody - corpo da tabela
tfoot - rodapé da tabela 

<table>
    <thead>
        <tr>
            <th>Estado</th>
            <th>População</th>
        </tr>
    <thead>
    <tbody>
            <tr>
                <td>São Paulo</td>
                <td>45.000.000</td>
            </tr>
            <tr>
                <td>Minas Gerais</td>
                <td>45.000.000</td>
            </tr>
    <tbody>
    <tfoot>
        <tr>
            <th>Total de Habitantes</th>
            <td>0000000000</td>
        </tr>
    <tfoot>
</table>
